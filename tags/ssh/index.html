 Posté le 12 August 2015  |  Richard Fagot

  1. Suivre les étapes indiquées dans Github sans faire la 5 ;
  2. Faire ssh-keyscan -t rsa github.com > ~/.ssh/known_hosts ;
  3. Faire l'étape 5 ;
  4. cloner le projet git en utilisant le clone SSH.
